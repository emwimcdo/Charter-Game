Charter Game

Controls

  • Use the dropdown menu to change functions.
  • While in drag, you can use the mouse to move the canvas, and the + - keys to zoom in and out, no matter what tool mode.
  • While in color hex, use the sliders to adjust the color and click a hex to color it. To pick a hex color, hold shift. Alt can be used at any time to temporarily switch to drag.
  • While in add node, type in your text, then click attach. Use the up and down arrow keys to adjust text size. Click default size button to reset. When the text is in the right location, click to add. If you made a mistake, press CTRL + Z to undo.

Joining a Room

  • Click the join/create room button.
  • Type in a room code.
  • Anyone else who types in that room code will join the room. Things added by you before the room is created will glitch. Please use a clean room.
